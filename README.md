# Support for Meritocracy and Access to Higher Education
## A code repository for the statistical analyses

Hye Won Kwon and Jani Erola

DOI here

## Description

Repository for the codes and country-level data file used for the analyses of Kwon, Hye Won and Jani Erola. 2025. “Support for Meritocracy and Access to Higher Education.” (A link to the article will be provided after the article is published.) All the analyses in the paper were conducted using STATA 18.

This study analyzed the ISSP 2009 data using multilevel analysis (i.e., random slope models). The country-level statistics (cohort-specific proportion of tertiary education, Gini index, and GDP per capita) were taken from OECD and World Bank databases. The final study sample contains 30,135 respondents from 32 countries. 

This reproducibility package contains two STATA do files and one data file(.dta).

- “KwonJani2025_MainAnalysis_OS.do”: STATA codes for the main analysis
- “KwonJani2025_Appendix_OS.do”: STATA codes for the robustness check analysis in presented in Appendix A
- “KwonJani_2025_MacroData.dta”: country-level data that contain cohort-specific proportion of tertiary education drawn from OECD (2023) and the Gini index and GDP per capita drawn from World Bank (2023)

The main survey data, the ISSP 2009 (International Social Survey Programme; ZA5400; Version 4.0.0), are publicly available via the GESIS portal (https://www.gesis.org/en/issp/data-and-documentation/social-inequality/2009).
